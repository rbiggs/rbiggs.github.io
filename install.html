Github

You can get the source and code ready to build with from Github. When you down the source code from Github, everything you need to get going is in the folder "dist". You do not need to run a build script. Also the source code includes a folder called "examples". In it you will find working examples of layouts and widgets that Truck provides.

If you have some specialized needs, you can build a version of Truck as needed. To build you need to run npm install on the folder:

npm install

After than you can build Truck in one of four ways:

  1. The complete framework
  2. Just the selector engine
  3. Just the MVC components
  4. Everyting except the selector engine

To build the whole thing, just run gulp. If you want to build just Truck's selector engine to use it as a light and modern replacement for jQuery, run:

gulp --wheels

To build just the MVC components, because you don't care about the layouts and widgets and maybe you want to use it with jQuery for desktop browsers, run:

gulp --mvc

And to build the framework but with Truck's DOM engine because you want to use jQuery, run:

gulp --engine

Please note that when you build a version without Truck's DOM engine, you need to load jQuery in the browser first. Also, you can use jQuery with the complete build as well, just load it first and Truck will use jQuery instead of its own DOM library.
