Gestures

Intro

Truck provides gestures to make user interactions more natural. These work identically with mouse or touch. Truck provides the following gestures:

  • tap
  • doubletap
  • longtap
  • swipeleft
  • swiperight
  • swipeup
  • swipedown

On Microsoft platforms, they use long taps for showing the context menu, so mostly this does not respond as expected. Otherwise these work the same across platforms and devices.

We strong recommend use Truck's tap for user interactions. Using a click has a rather long delay, and using a touchstart and cause scrollable lists to behave unexpected do to triggering actions while trying to scroll. A tap is the sweet spot for most user interactions.
