Utilities

Use

Truck provides a number of utility functions for you to use. The first set are as follows:

  • $.isEmptyObject()
  • $.isInteger()
  • $.isFloat()
  • $.encode()
  • $.escapeHTML()
  • $.concat()
  • $.uuid()

$.isEmptyObject(object) tests where an object is {} or where it has properties. This is a boolean value.

$.isInteger(number) tests whether the number is an integer or not. This is a boolean value.

$.isFloat(number) tests whether the number is a float or not. This is a boolean value.

$.encode(data) returns the value from a form element encoded.

$.escapeHTML(data) escapes the html tags, parenthesis, etc.

$.concat(args) takes a series of strings and returns one string. This is useful where you need to concatenate a lot of string pieces together

var str = $.concat('This', ' ', 'is', ' ', 'a' , ' ', 'string', '.');

$.uuid() create a uuid. This is a string. You can use it to create a unique identifier for objects in a collection, or for ids of DOM elements. Truck uses $.uuid internally for this purpose.

The Haz Helpers

These help you avoiding writing loops to handle finding elements by selectors. These do not return booleans like their jQuery cousins. Instead they directly return the elements that match the selector. This allows you to write onelines to access them.

  • $(selector).iz(selector)
  • $(selector).iznt(selector)
  • $(selector).haz(descendant)
  • $(selector).haznt(descendant)
  • $(selector).hazClass(class)
  • $(selector).hazntClass(class)
  • $(selector).hazAttr(attribute)
  • $(selector).hazntAttr(attribute)

Here are some examples using them:

$('#list').iz('ol').css('border', 'solid 1px blue');
$('.list').iznt('.disabled').show();
$('#list').haz('li').addClass('important');
$('#list').haznt('li').append('<li>Some default text</li>');
$('#list').hasClass('important').css({'background-color': 'blue, 'color': '#fff''});
$('button').hazAttr('disabled').removeProp('disabled');

Truck lets you iterate over element collections using a normal JavaScript foreach function. jQuery uses the backward each function where the index value comes first. Since most of the time we care about the context and not the index, the foreach gives us an option.

$('li').forEach(function(li) {
  console.log(li.nodeName);
});

Truck provides two function to enable and dispable form elements:

  • $(input).enable()
  • $(input).disable()
