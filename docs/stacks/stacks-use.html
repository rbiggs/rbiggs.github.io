Stacks

Use

Stacks are abstractions for arrays and collections of objects. Why would you want to use an abstraction of an array? To protect your data from accidental modification. And, because Stacks are optimized for performance, executing the same operations as native functions at faster speeds.

To make an empty stack, or pass an array to it:

// Create an empty stack:
var myStack = $.Stack();

// Create a stack of numbers:
var nums = [1,2,3,4,5];
var numStack = $.Stack(nums);

// Create a stack of objects:
var names = [
  {name: 'Wobba'},
  {name: 'Joe'},
  {name: 'Sam'}
];
var nameStack = $.Stack(names);

Once you have data in a stack, you can run various stack methods on it:

// Iterate over a stack:
myStack.forEach(function(item) {
  console.log(item);
});

// Filter a stack for items that cost $100:
var result = myStack.filter(function(item) {
  return item.price = 100;
});

// Add a new item to a stack:
myStack.push({name: 'Sally'});

remove the first item from the stack:
var firstPerson = myStack.shift();
