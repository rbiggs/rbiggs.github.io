Everything You Need to Know

Stacks

API

size

size()

Get the size (length) of the stack.

push

push(data)

Push data onto the stack.

pop

pop()

Pop the last item off of the stack.

eq

eq(index)

Get an items from the stack by its index. This is a zero-based number.

forEach

forEach(callback)

Run a loop over the items in the stack. This takes a callback with the content and index as values, just like foreach in arrays.

each

each(callback)

This is the same as above but uses the reverse order for the index and content like jQuery.

shift

shift()

Remove the first item from the stack.

unshift

unshift(data)

Add a new item to the front of the stack.

slice

slice()

Get a slice from a stack. This works just like slice on an array.

splice

splice()

Splice a stack. This works just like splice on an array.

sort

sort()

Sort a stack. This works just like sort on an array.

sortBy

sortBy(args)

Sort a stack by the provided properties. You can provide multiple properties to sort by. If a property is prefixed with a hyphen "-" the property will be sorted in descending order instead of ascending. order

filter

filter(callback)

Filter a stack. This works just like filter on an array.

map

map(callback)

Map the contents of a stack. This works just like map on an array.

join

join(separator)

Join the contents of a stack. This works just like join on an array.

concat

concat(data)

This concats an array to a stack.

reverse

reverse()

Reverse a stack. This works just like reverse on an array.

every

every(callback)

Sort a stack. This works just like every on an array.

some

some(callback)

Run a callback on a stack. This works just like some on an array.

unique

unique()

This removes any dupicates from a stack.

purge

purge()

This sets the content of the stack to [].

getData

getData()

This returns the entire contents of the stack as an array.
