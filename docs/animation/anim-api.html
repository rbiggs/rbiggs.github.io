Everything You Need to Know

Anim

API

$.anim

$.anim(element)

This method takes either a selector, a plain DOM node or a Truck DOMStack. In all cases, $.anim expects a single element. If you pass in a Truck element collection, will only use the first.

This method is how you initialize an animation sequence. All the other animation methods below can be chained to this method.

// Pass in a string selector:
$.anim('#myElement');

// Pass in a node:
var el1 = document.querySelect('#myEl1');
$.anim(el1);

// Pass in a Truck element:
var el2 = $('#myEl2');
$.anim(el2);

When you initialize an animation, Truck gets the styles of the element to use as the initial state of any properties that need to be animated.

set

set(property, value)

This is the most versatile method. It allows you to set any property on an element for animation. It takes two arguments, a property and a value for the property. Both of these should be quoted strings. The property and the value should be in normal CSS format:

$.anim('#element')
.set('background-color', '#ff0000')
.set('font-size', '20pt')
.set('color', 'green')
run();

add

add(property, value)

This method allows you to increase the value of the provided property. The second argument is a number. This will get converted to a pixel value. As such this can only be used to increase the value of properties that can accept pixel values.

// Increase the left margin:
$.anim('#el')
.add('margin-left', 300)
run();

sub

sub(property, value)

This method allows you to decrease the value of the provided property. The second argument is a number. This will get converted to a pixel value. As such this can only be used to increase the value of properties that can accept pixel values.

// Decrease the absolute left position:
$.anim('#el')
.sub('left', 300)
run();

x

x(number)

This method translates the element along its x coordinate. The number value will be converted to pixels.

// Translate the element on its x axis:
$.anim('#el')
.x(300)
run();

y

y(number)

This method translates the element along its y coordinate. The number value will be converted to pixels.

// Translate the element on its y axis:
$.anim('#el')
.y(200)
run();

translate

translate(number[, number])

This method translates the element along its x and y coorinates. The number value will be converted to pixels. If only one number is provided, the element will only be translated along its x axis.

// Translate the element on its x & y axis:
$.anim('#el')
.translate(300, 200)
run();

// Translate it only on the x axis:
$.anim('#el')
.translate(300)
run();

rotate

rotate(number)

This method rotates an element. It takes a number that will be converted to degrees.

// Rotate element:
$.anim('#el')
.rotate(45)
run();

// Rotate element backwards:
$.anim('#el')
.rotate(-90)
run();

scale

scale(number[, number])

This method allows you to scale an element up or down. 1 is the default size, 2 is double, .5 is 50%, etc. The first is the width and the second is the height. If only want value is provided, the element is scaled proportionately.

// Scale the element up:
$.anim('#el')
.scale(3, 5)
run();

// Scale the element down:
$.anim('#el')
.scale(.5, .75)
run();

// Scale the element up uniformly:
$.anim('#el')
.scale(2)
run();

// Scale the element down uniformly:
$.anim('#el')
.scale(.25)
run();

scaleX

scaleX(number)

Scale a element by its width.

// Scale up the element's width:
$.anim('#el')
.scaleX(2)
run();

// Scale down the element's width:
$.anim('#el')
.scaleX(.5)
run();

scaleY

scaleY(number)

// Scale up the element's height:
$.anim('#el')
.scaleY(3)
run();

// Scale down the element's height:
$.anim('#el')
.scaleY(.25)
run();

skew

skewX(number[, number])

This method allows you to skew an element by its x and y axis. If only one value is provided then the element is only skewed on its x axis.

// Skew the element on both axis:
$.anim('#el')
.skew(100, 90)
run();

// Skew the element on its x axis:
$.anim('#el')
.skew(200)
run();

skewX

skewXs(number)

This method skews an element by its x axis:

// Skew element on its x axis:
$.anim('#el')
.skewX(200)
run();

skewY

skewY(number)

This method skews an element by its y axis:

// Skew element on its y axis:
$.anim('#el')
.skewY(100)
run();

matrix

matric(numbers)

If you're comfortable with transform matrices, you can use this method to set one on an element:

// Set a matrix on the element:
$.anim('#el')
.matrix(1, -0.00000000000000024492935982947064, 0.00000000000000024492935982947064, 1, 0, 0)
run();

duration

duration(string)

This method lets you set the duration of an animation sequence. B default all animations last 500 milliseconds. You can use this method to change that. If you set up multiple then sequences in your animation, each one can have their own duration. You can pass this method a number in milliseconds, or a string with an length identifier. These are typical time in string format as expected for CSS transtion duration:

$.anim('#el')
.duration(2000)
.add('left', 300)
run();

// With a string time:
$.anim('#el')
.duration('2s')
.add('left', 300)
run();

ease

ease(easingName)

This method allows you to add easing to an animation sequence.

  • in
  • out
  • in-out
  • snap
  • linear
  • ease-in-quad
  • ease-out-quad
  • ease-in-out-quad
  • ease-in-cubic
  • ease-out-cubic
  • ease-in-out-cubic
  • ease-in-quart
  • ease-out-quart
  • ease-in-out-quart
  • ease-in-quint
  • ease-out-quint
  • ease-in-sine
  • ease-out-sine
  • ease-in-out-sine
  • ease-in-expo
  • ease-out-expo
  • ease-in-out-expo
  • ease-out-in-back
  • ease-in-circ
  • ease-out-circ
  • ease-in-out-circ
  • ease-out-in-circ
  • ease-in-back
  • ease-out-back
  • ease-in-out-back
$.anim('#el')
.duration('2s')
// Add easing:
.ease('ease-out-expo')
.add('left', 300)
run();

You can also use matrix numbers to define a custom easing function. To do that you must enclose the cubic bezier fuction in quotes:

$.anim('#el')
.duration('2s')
// Add custom easing:
.ease('cubic-bezier(-.25,1,1,-.25)')
.add('left', 300)
run();

delay

delay(milliseconds)

This method lets you define a delay before the animation runs. It takes a value in millisecons, or you can provide a string with a length identifier as used in CSS transition duration.

$.anim('#el')
.duration('2s')
.delay(2000)
.add('left', 300)
run();

// Use a string value for delay:
$.anim('#el')
.duration('2s')
.delay('1s')
.add('left', 300)
run();

anim

anim(element)

This method allows you to choose another element to animate as part of the sequence. This could be the same element, in which case this would allow you to completely change its state before the next sequence of transitions, or you could provide another element. this allows you to have one animation set up to animate multiple elements.

$.anim('#el')
.duration('2s')
.delay('1s')
.add('left', 300)

// Animate another element:
.anim('#el2')
.duration('1s')
.sub('margin-left', 300)
run();

If you are having problems where it seems like your sequences are not executing as expected, you might try breaking it up using .anim targeting the same element.

then

then(fn)

This method allows you to two things, return the animation so that you can chain on another sequence, or pass in a callback that will be deferred untill the previous sequence ends. When there is more than one then block in an animation, each additional then will require a final pop method to terminate it. See pop below.

// Define a then block with another sequence:
$.anim('#el')
.duration('2s')
.delay('1s')
.add('left', 300)
.then()
  .set('color', 'red')
  .set('font-size', '30px')
run();

// Pass callback to then:
$.anim('#el')
.duration('2s')
.delay('1s')
.add('left', 300)
.then(function() {
  console.log('Just a message here.');
})
  .set('color', 'red')
  .set('font-size', '30px')
run();

pop

pop()

This method is used to signal the termination of a then sequence. If your animation has only one then block, the run method at the end will terminate it. If you have more than one then, for each additional then you will need to provide a pop method at the end of the animation:

// Define a then block with another sequence:
$.anim('#el')
.duration('2s')
.delay('1s')
.add('left', 300)
// First then, no pop need:
.then()
  .set('color', 'red')
  .set('font-size', '30px')
// Second then, pop need:
.then()
  .x(-200)
// Thrid then, pop need:
.then()
  .translate(300, 200);
  .easing('snap')
  .duration('3s')
// End second then:
pop()
// End third then:
pop()
run();

run

run(fn)

This method lets you run your animation. Optionally you can pass it a callback. If your animation consists of a single sequence, the callback will be execute when it terminates. However, if you have used then blocks or .anim methods to create nested sequences, the callback passed to run will execute when the first sequence ends. In the case where you have multiple sequences and want to execute a callback, pass it to the last then block.

$.anim('#el')
.set('color', 'red')
.run();

// Pass run a callback:
$.anim('#el')
.set('color', red')
.run(function() {
  alert('This animation just ended!');
});

defaults

$.anim.defaults

This is used to set default duration for your animations. Its value is set to 500 milliseconds. This is shared by all animations unless you use the duration method to set a duration for a sequence. You can change the default as follows:

$.anim.defaults = {
  duration: 2000
};
