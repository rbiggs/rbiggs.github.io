Dispatcher

API

receive

$.receive(handle, callback)

This method sets up a receiver. This takes two arguments, a handle to listen to and a callback to execute when the handle is dispatched. The callback gets passed two arguments when executed: the handle and any data sent by the dispatcher. You can setup a receive by assigning it to a variable. Doing so allows you to cancel the receiver if you choose to. Without the variable reference, there is no way identify a receiver for cancelation.


var myReceiver = $.receive('my-handle', function(handle, data) {
  console.log('The data received is: ' + data);
});

dispatch

$.dispatch(handle, data)

This method lets you dispatch a handle and possible data to a receiver.


// Dispatch to the receiver defined above:
$.dispatch('my-handle, 'This is a message.');

cancelReceiver

cancelReceiver(receiverName)

This method allows you to cancel a receiver. When you cancel a receiver, it will be permanently disabled.


$.cancelReceiver('myReceiver');
