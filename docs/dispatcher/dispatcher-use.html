Dispatcher

Use

Truck's Dispatcher is an evented bus. You can use this to set up a basic PubSub type of interaction. The Dispatcher has receivers and dispatchers. Although you don't have to, you should probably name your receivers. You do this by assigning them to a variable. That way, if you need to later get rid of a receiver you can do so using the variable. Box receivers and dispatchers you a handle. This is a string value. You can use any type of delimiters to create a namespace if that is helpful, or just use a simple name.

$.receive()

Receivers take two arguments, the handle their using and a callback. If the dispatcher sent any data, this will be available to the callback as the first argument:

var newsUpdater = $.receive('news/update', function(data) {
  $('#news').append('<li>' + data + '</li>');
});

If the data being received was an array, we could loop over it:

var newsUpdater = $.receive('news/update', function(data) {
  data.forEach(function(item) {
    $('#news').append('<li>' + item + '</li>');
  });
});

You should be aware of what type of data you are expecting with a dispatch. Or you can set up different receivers that test the data before acting. Then only the receiver that is expecting a particular type of data will handle its output.

$.dispatch()

To make a dispatcher, you use the $.dispatch() method. This takes two arguments, the handle to dispatch to and any data. You do not have to dispatch data. Similarly, a receiver does not need to get data. Both of these could simply be about dispatching the handle. In this case the dispatcher is acting like a custom event system.

Below is an example of a dispatcher for our previous reeiver example:

// Dispatch a single item:
$.dispatch( 'news/update', 'We need piles of money now!' );
// Dispatch an array of data:
$.dispatch( 'news/update', [{'news-flash':'Not much has happened yet.'},{'update':'More of the same.'}] );

$.cancelReceiver()

You can cancel a recevier at any time. Please note that doing so permanently disables it. If you want to be able to stop a receiver and start it later, then you probably want to set up a mediator instead. Please see Mediators for more information.

Below is an example of canceling a receiver:

$.cancelReceiver( newsUpdater );
