Demos

We currently have two reference apps for you to download in order to see how to put TruckJS's layouts, widgets, model, views and routes together to make an app.

  • SFCoffee
  • Vino
  • Fragranz
  • TodoMVC

TodoMVC is our take on the popular TodoMVC project. But unfortunately it is designed as a desktop app. Since TruckJS is all about the mobile experience, we create a mobile version of TodoMVC where the UI reflects what a mobile app should be.

You can see these apps live on your mobile devices at these urls. Please note that they are using the iOS theme:

  • Vino
