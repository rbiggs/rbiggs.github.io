Extend

Truck provides a simple and convient way to extend objects, $.extend(). This lets you add one object's properties to another. When copying properties, if any already of the source object's properties already exist on the target, these will be replaced. If this is not the behavior you want, look at using $.Mixin instead.

Below is a basic object literal:

var sam = {
  name: 'Sam',
  job: 'developer',
  hobby: 'twiddling thumbs',
  address: '1234 Lane, SomeWhereVille'
}

And we want to extend it with the following object:

var newProps = {
  job: 'cook',
  address: '000 DeadEnd, NoPlace',
  age: 1000
};

To extend sam with newProps, we pass sam followed by newProps to R.extend:

$.extend(sam, newProps);

This results in sam being updated with newProps. sam's job and address are updated to that of newProps, and sam gets a new property: age.

{
  name: 'Sam',
  job: 'cook',
  hobby: 'twiddling thumbs',
  address: '000 DeadEnd, NoPlace',
  age: 1000
}

By the way, you can also just define properties inline as the second argument:

$.extend(sam, {age: 1000, job: 'janitor'});
