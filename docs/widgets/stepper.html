Stepper

A stepper is a small control that allows the user to increase or decrease a value between predetermined values. It's not a picker. It's for when the different between possible choice is not great. The user decreases or increase the value by tapping on either side of the stepper.

To create a stepper you use a span with the class "stepper":

<span class='stepper' id="digits"></span>

To initialize a stepper, we use the $.Stepper() method and pass it three options: min value, max value and default value:

var myStepper = $.Stepper({ 
  element: '#digits', 
  min: 1,
  max: 8,
  defaultValue: 3
});

We can get the current value of the stepper at any time by running getValue() on the stepper instance:

var stepperValue = myStepper.getValue();

Take a look at the file "stepper.html" in the examples folder in the source code from Github.
